# Описание проекта и задача
Из данных оператора мобильной связи по поведению пользователей нужно построить модель для задачи классификации, которая выберет подходящий тариф.

## Данные:
Одна таблица с следующими данными:

* сalls — количество звонков,
* minutes — суммарная длительность звонков в минутах,
* messages — количество sms-сообщений,
* mb_used — израсходованный интернет-трафик в Мб,
* is_ultra — каким тарифом пользовался в течение месяца («Ультра» — 1, «Смарт» — 0).
## Стек
Pandas, Seaborn, Sklearn

## Результат
Подобрана модель классификации для рекомендации тарифов клиентам, тест на адекватность пройден.
